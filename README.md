

自定义 cli 

用于快速创建 Vue 项目

1. Vue H5
2. Vue NuxtJS PC


## Install

```
npm i -g pvue
```

## Usage

```
pvue init
```


# 其他

1. `commander`: nodejs 命令行工具
2. `inquirer`: 命令行交互(收集参数)
3. `shelljs`: 在代码中执行 shell 命令
4. `log-symbols`: 给 log 添加 icon
5. `ora` loading 动画
6. `node-notifier` 通知
7. `download-git-repo` 下载 git 项目
